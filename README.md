# insper-salas-livres

Criei esse projeto durante o 5º semestre da minha graduação de Engenheria da Computação no [Insper](https://insper.edu.br/), pra ajudar a comunidade a encontrar salas desocupadas para estudar. Já me formei, então resolvi deixar a codebase aberta pra que outros alunos possam contribuir ou fazer forks :)

> [!CAUTION]
> Esse projeto foi criado com base em um endpoint de API do Insper não documentado, tirado [da página de agenda](https://cgi.insper.edu.br/Agenda/) (abre, aperta F12 e dá uma espiada na aba de Rede). 
> 
> Embora a página e o endpoint sejam públicos, tome cuidado para não spammar ele e acabar deixando alguém do TI bravo.

## Como rodar o projeto

Usando o seu package manager favorito, instale as dependências e rode o servidor de desenvolvimento (eu uso o [Bun](https://bun.sh/))

```bash
bun install
bun run dev
```
